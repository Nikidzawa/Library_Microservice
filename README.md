# Library_Microservice
Полностью рабочий рест апи микросервис для библиотек.

Клиент и база данных изолированы в докер-контейнерах.

Присутствует документация на Swagger, все ошибки обработаны.

Полный стек:
* Java 21
* Spring Boot
* Docker
* Swagger
* Hibernate
* postgresql
* Lombok
